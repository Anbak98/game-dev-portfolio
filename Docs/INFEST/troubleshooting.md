# ⚠️ 트러블슈팅 - 몬스터 이동 속도가 컴퓨터마다 다른 문제

### 1. 발단

(1) 팀원들과 빌드 테스트를 진행하던 중, 화면 공유를 통해 본 팀원의 좀비 이동 속도에서 위화감을 느꼈습니다.  
(2) 다른 팀원들의 화면도 살펴보았고, 팀원들간 좀비 이동 속도에 편차가 있음을 발견했습니다.  
(3) 처음에는 프레임보정이 적용되지 않았을 거라 생각해 몬스터 이동 로직을 검토했습니다.  
(4) 그러나 몬스터 이동에 사용 중이던 NavMesh.SetDestination() 함수는 기기별 에이전트 속도 편차와 같은 이슈가 발견되지 않았습니다.
