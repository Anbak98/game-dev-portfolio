# 프로젝트 #1 


### 게임 "INFEST" - 친구들과 함께 시원하게 총을 쏘며 스트레스를 푸는 게임
### 기술 스택 : Unity / Photon Fusion 2  
### 작업 개요
- 애니메이션 트리거, 피격 여부 등 **ON/OFF 상태 플래그**를 NetworkBool로 단일 비트 동기화하여 데이터 전송 최적화 및 일관성 확보.
- 대규모 좀비 웨이브 환경에서는 Shared 토폴로지의 상태 불일정을 방지하기 위해 **Host 토폴로지**를 적용, 네트워크 시뮬레이션을 단일 주체에서 처리하여 게임 상태 완전 일관성 유지.
