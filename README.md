# 프로젝트 #1 


### 게임 "INFEST" - 친구들과 함께 시원하게 총을 쏘며 스트레스를 푸는 게임
### 사용 기술 : Unity / Photon Fusion 2  
### 작업 개요 : 
1. 애니메이션 트리거, 피격 여부 등 ON/OFF 상태 플래그는 Photon Fusion 2의 NetworkBool을 활용해 단일 비트로 동기화하여 데이터 전송 최적화 및 일관성 확보.
2. 대규모 좀비 웨이브 환경에서는 Shared 토폴로지가 상태 불일정을 초래할 수 있어, Host 토폴로지를 적용하여 네트워크 시뮬레이션을 단일 주체에서 처리함으로써 게임 상태의 완전한 일관성을 유지.
